## ⚡ Quick Links for Common Tasks

### Development
- Start dev server: `npm run dev` (see QUICK_START.md)
- View dev server: http://localhost:3000
- Stop dev server: Ctrl+C

### Customization
- Change hero text: Edit `src/components/sections/Hero.tsx`
- Update skills: Edit `src/components/sections/About.tsx`  
- Add projects: Edit `src/components/sections/Projects.tsx`
- Change colors: Edit `tailwind.config.ts`

### Deployment
- Build: `npm run build` (see COMMANDS.md)
- Deploy to Vercel: Push to GitHub (see README.md)

### Troubleshooting
- See issues? Check [README.md](README.md) Troubleshooting section
- Need terminal commands? Check [COMMANDS.md](COMMANDS.md)

**Beginner (First Time)**
1. QUICK_START.md - 3 minutes

4. Edit a component and see changes

**Intermediate (Customizing)**
1. Review QUICK_START.md "Where to Customize"
2. Open component file (e.g., Hero.tsx)
3. Read inline comments
4. Make changes (Ctrl+S to save)

**Advanced (Deploying)**
1. Run `npm run build`
2. Check README.md Deployment section
3. Follow Vercel or Netlify instructions
4. Deploy!
